\# Digital Twin Advisor Agent Architecture



The Digital Twin Advisor Agent helps users plan digital twin implementations.



Workflow:



User Question  

↓  

Digital Twin Advisor Agent  

↓  

LPI Tool Calls

\- query\_knowledge

\- get\_case\_studies

\- get\_insights  

↓  

Processing with SMILE methodology  

↓  

Recommendation Output



The agent combines knowledge entries, case studies, and expert insights

to generate explainable recommendations.
